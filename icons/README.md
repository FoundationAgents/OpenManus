# PWA Icons

This directory contains all the required icon files for your Progressive Web App.

## Generated Icons

### Standard Icons
- `icon-72x72.png` - Small Android icon
- `icon-96x96.png` - Android notification icon
- `icon-128x128.png` - Chrome Web Store icon
- `icon-144x144.png` - Microsoft tile icon
- `icon-152x152.png` - iOS home screen icon
- `icon-192x192.png` - Android home screen icon (minimum)
- `icon-384x384.png` - Android splash screen
- `icon-512x512.png` - High-resolution icon (recommended)

### Special Purpose Icons
- `maskable-icon-512x512.png` - Adaptive icon with safe zone for Android
- `home-96x96.png` - Shortcut icon for quick actions

## Icon Design

All icons feature:
- Modern gradient background (purple to pink)
- White rounded square container
- "P" symbol for PWA branding
- High contrast for visibility
- Optimized for all display sizes

## Preview

Open `preview.html` in your browser to see all icons and verify they display correctly.

## Customization

To customize these icons:
1. Edit the `generate_icons.py` script (if available)
2. Modify colors, shapes, or symbols
3. Re-run the script to regenerate all sizes
4. Or use your own design tool and replace these files

## Browser Support

These icons are compatible with:
- Chrome/Edge (Android & Desktop)
- Safari (iOS & macOS)
- Firefox
- Opera
- Samsung Internet
